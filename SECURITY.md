# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public GitHub issue
2. Send a private report via [GitHub Security Advisories](https://github.com/Nowohier/CN_HashiWpf/security/advisories/new)

You should receive a response within 7 days. Thank you for helping keep this project secure.
